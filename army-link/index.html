Passkey is hashed locally (Web Crypto SHA-256) and compared against a stored digest. The plaintext passkey never leaves this device, is never logged, and is never persisted. Unlock state is stored only in sessionStorage for this tab.
Soft-gate disclaimer. This is a static GitHub Pages site. The passkey check runs entirely in your browser and can be bypassed by anyone willing to read the source. Treat it as a friction layer, not as access control. No secrets, credentials, or trading capabilities are exposed behind this gate.
